## Settings
### Create a virtual environment (recommended)
```bash
python -m venv venv
.\.venv\Scripts\activate
```
### Install library
```bash
pip install -r requirements.txt
```
### Run app with virtual environment (Personal computer - Command Prompt)
```bash
streamlit run .\stress_dashboard_app.py
```
